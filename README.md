
# pterodactyl-turkce

Sunmuş olduğum bu yazılım, Pterodactyl panelinizi tek bir komut ile Türkçe yapmanıza yarar.


## Kurulum

⭐ Sunmuş olduğum Türkçeleştirme scriptini kurmak için aşağıdaki komutları çalıştırmanız yeterlidir.

```bash
wget https://shaigannn.com.tr/shell/pterotr.sh
chmod +x pterotr.sh
./pterotr.sh
```
